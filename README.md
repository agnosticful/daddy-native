# daddy-native

Android/iOS client of D.A.D.D.Y on React Native.

## Development

We use [Expo](https://expo.io/).

1. Make sure if Node is installed on your machine and `npm i`
2. Install Expo app ([Android](https://play.google.com/store/apps/details?id=host.exp.exponent) / [iOS](https://itunes.apple.com/app/apple-store/id982107779)) to your phone
3. `npm start`
4. Scan the barcode on your terminal with the Expo app
